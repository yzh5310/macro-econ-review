📋 核心公式速查（6个必须记住）零基础可看

每个公式都配了白话解释和记忆技巧。不需要死记硬背，理解含义后自然记得住。

① GDP 支出法 最基础

Y = C + I + G + NX
白话：一个国家一年生产的东西，要么被老百姓买了（C消费），要么被企业买了（I投资），要么被政府买了（G政府支出），要么卖给外国人（NX净出口）。
记忆：谐音"吃（C）一（I）个（G）馕（NX）"。
考试：这是宏观经济学的"万能公式"，几乎每个模型都从它出发。

② 投资乘数 高频计算

k = 1 / (1 − MPC)
白话：MPC是"每多赚1块钱，你愿意花多少"。如果MPC=0.8（花8毛存2毛），乘数k=5，意思是投资增加100元，最终总收入增加500元。
为什么？你花80元→别人赚80元→别人花64元→又有人赚64元……像滚雪球一样。
常见错：MPC越大，乘数越大（不是越小！）。MPC=0.9时乘数高达10。

③ GDP平减指数 概念题

GDP平减指数 = (名义GDP / 实际GDP) × 100
白话：名义GDP是"按今年价格算的"，实际GDP是"按去年价格算的"。两者一比就知道今年物价涨了多少。
例子：今年生产了同样多的东西，名义GDP=110万，去年价格算只有100万，那平减指数=110，说明物价涨了10%。

④ 货币需求函数 IS-LM核心

L = kY − hr
白话：人们想持有多少钱？收入越高需要越多钱（kY），利率越高需要越少钱（−hr，因为利率高不如去买债券）。
记法："收入多了想花钱，利率高了想存钱"。

⑤ 菲利普斯曲线 需要理解

π = πᵉ − β(u − uₙ)
白话：通胀率π取决于：预期通胀πᵉ、以及失业率u离自然失业率uₙ有多远。
通俗：失业低于自然率→经济过热→工资涨→物价涨（通胀高）。反之亦然。
考试：短期：失业和通胀此消彼长。长期：失业回自然率，通胀取决于预期。

⑥ 索洛稳态条件 经济增长

s·f(k*) = (δ + n + g)k*
白话：每个人存的钱（s·f(k)）刚好够用来补折旧(δ)、分给新工人(n)和技术进步(g)。
通俗：像水池——进水（储蓄投资）= 出水（折旧+人口增长+技术进步）时，水位稳定。
核心结论：提高储蓄率能提高收入水平，但不能提高长期增长率。长期增长只靠技术进步！
💡 公式怎么记？ 不要一个字母一个字母地背！先理解公式在说什么（上面的白话解释），再记住关键数字关系（比如MPC=0.8→乘数=5）。考试时先写公式再代入数字，步骤分不会丢。

高频概念速查

GDP 衡量的是一定时期内最终产品和服务的市场价值，但它不反映收入分配、闲暇、环境质量、地下经济和非市场活动，因此不能完全等同于社会福利。
扩张性财政政策增加政府支出，可能推高利率，导致私人投资减少，这种私人投资被政府支出"挤出"的现象称为挤出效应。
当利率已经很低时，人们愿意持有货币而不愿购买债券，货币需求对利率极其敏感，此时增加货币供给可能难以继续降低利率，货币政策效果减弱。
滞胀是经济停滞、高失业与通货膨胀同时存在的状态，通常可由负向供给冲击解释，例如生产成本上升导致 SRAS 左移。
自然失业率包括摩擦性失业和结构性失业，是经济正常运行中仍会存在的失业；周期性失业则与经济衰退和总需求不足有关。

核心概念精讲（不翻书搞定全章）

按章节顺序逐层展开，每一块都包含定义、公式、图示和应试要点。折叠展开即可快速查阅。

🗺️ 这章讲什么？ 就像体检要量身高体重血压一样，衡量一个国家的经济"健不健康"也需要指标。这章介绍三个最重要的"体检指标"：GDP（经济总量）、CPI（物价水平）、失业率（就业状况）。

GDP（国内生产总值）

定义：一定时期内一国境内生产的所有最终产品和服务的市场价值总和。

白话版：GDP就是一个国家一年"赚了多少钱"。注意是"生产了多少"而不是"消费了多少"。比如你买了一台5000元的电脑，这5000元就贡献给了当年的GDP。
关键限制：GDP只算"最终产品"（卖给消费者的），不算"中间产品"（面粉→面包，只算面包的价格）。重复计算就乱了。

三种核算方法：

  • 支出法：GDP = C（消费）+ I（投资）+ G（政府购买）+ NX（净出口）
  • 收入法：GDP = 工资 + 利息 + 租金 + 利润 + 折旧 + 间接税
  • 生产法：各行业增加值之和
GDP = C + I + G + NX
⚠️ 常见误区：
① GDP≠GNP！GDP按"境内"算，GNP按"国籍"算。中国企业在海外的产值算在中国GNP里，但不算在中国GDP里。
② GDP不含二手交易（房子转手不算）、不含非法活动、不含家务劳动。
③ GDP高≠人民幸福！它不反映贫富差距、环境污染、闲暇时间。

名义GDP vs 实际GDP

名义GDP：按当年价格计算。
实际GDP：按基年价格计算，剔除价格变动。

GDP平减指数 = (名义GDP / 实际GDP) × 100
通俗理解：假如一个国家去年生产了100个面包，今年还是100个，但面包从10元涨到11元。名义GDP从1000变成1100（看起来增长了10%），但实际GDP还是1000（实际产量没变）。GDP平减指数=110，说明物价涨了10%。
应试要点：计算"经济增长率"时用实际GDP，而非名义GDP。平减指数衡量的是全部商品价格变化。

CPI vs GDP平减指数

对比维度CPIGDP平减指数
商品范围一篮子消费品所有国内产品
进口品包含不包含

记忆技巧：CPI是"老百姓视角"——只看消费品，进口也算（因为你要买进口手机）；GDP平减指数是"国家视角"——只看国内生产的东西。

失业率与奥肯定律

自然失业率：摩擦性失业 + 结构性失业（经济正常运转时的最低失业率）。

周期性失业：经济衰退导致的总需求不足引起的失业。

三种失业一句话版：
• 摩擦性：辞职后在找新工作（正常换工作的间隙期）
• 结构性：你的技能被淘汰了（比如打字员被电脑取代）
• 周期性：经济不好公司裁员（经济衰退导致）
奥肯定律：实际GDP每低于潜在GDP 2%，失业率上升约1个百分点。即：Δ失业率 ≈ −0.5 × (实际GDP增长率 − 潜在GDP增长率)
🗺️ 这章讲什么？ 回答一个核心问题：一个国家的总收入是怎么决定的？ 答案是：由总需求（大家愿意花多少钱）决定的。花得越多，生产越多，赚得越多——但有个"乘数放大器"效果。

消费函数与储蓄函数

凯恩斯消费函数：C = a + b·Yd（a:自主消费, b:边际消费倾向MPC）

MPC：每增加1单位可支配收入，消费增加多少（0 < MPC < 1）

MPS：边际储蓄倾向 = 1 − MPC

C = a + MPC × Yd　　S = −a + (1−MPC) × Yd
白话版：C = a + MPC × Yd 意思是：你的消费 = 基本生活费(a) + 收入的一部分(MPC × Yd)。
比如a=200（每月最低花200），MPC=0.8（每赚100元花80元），月收入5000元，那消费=200+0.8×5000=4200元。
MPC记住：MPC永远在0和1之间。MPC越大→越爱花钱→乘数越大。
⚠️ 常见误区：MPC越大，乘数越大（不是越小！）。MPC=0.8时乘数=5，MPC=0.9时乘数=10。别搞反了！

均衡收入与乘数

均衡条件：Y = AE = C + I + G + NX（总产出=总需求）

投资乘数：k = 1/(1 − MPC)

政府支出乘数：ΔY/ΔG = 1/(1 − MPC)

税收乘数：ΔY/ΔT = −MPC/(1 − MPC)（注意负号！减税使收入上升）

ΔY = k × ΔI = ΔI / (1 − MPC)
乘数效应通俗版：政府投资100万修路→建筑工人赚了100万→工人花掉80万(MPC=0.8)买家电→家电厂赚80万→家电厂工人花64万……
总共产生的收入 = 100/(1-0.8) = 500万！这就是"滚雪球"效应。
税收乘数为什么小？因为减税100元→你只花80元(MPC)→乘数是MPC/(1-MPC)=4，比政府直接支出的乘数5要小。直觉：减税的钱不一定全花出去。
应试要点：MPC越大，乘数越大。MPC=0.8时乘数=5。计算题通常给MPC让你算均衡收入和乘数。注意区分"自主消费a增加"和"MPC变化"的不同效应。
⚠️ 考试陷阱："自主消费a增加10"和"MPC提高0.1"效果完全不同！a增加只影响截距（水平移动），MPC变化影响斜率（曲线变陡）。考试经常考这个区别。
🗺️ 这章讲什么？ 上一章只考虑了"产品市场"（买东西），但现实中还有"货币市场"（存钱/借钱/利率）。IS-LM模型把两个市场放在一起，同时决定收入Y和利率r。
一句话记忆：IS管"商品买卖"，LM管"钱的借贷"，两条线交叉的地方就是均衡。

IS曲线（产品市场均衡）

含义：使产品市场均衡（I=S）的所有(Y, r)组合。

斜率：负斜率——利率r上升 → 投资I下降 → 总需求下降 → 均衡收入Y下降。

右移因素：G↑、T↓、自主消费↑、自主投资↑、NX↑

左移因素：G↓、T↑、自主消费↓、自主投资↓、NX↓

IS曲线白话版：IS是"投资=储蓄"的连线。利率高了→企业不愿借钱投资→收入少→储蓄也少。所以IS向右下方倾斜。
什么会让IS右移？政府花钱多了（G↑）或减税了（T↓）→总需求增加→同样的利率下收入更高→IS右移。

LM曲线（货币市场均衡）

含义：使货币市场均衡（L=M/P）的所有(Y, r)组合。

斜率：正斜率——收入Y上升 → 货币需求L↑ → 利率r上升以维持均衡。

右移因素：M↑、P↓（实际货币供给增加）

左移因素：M↓、P↑（实际货币供给减少）

货币需求：L = kY − hr　（k:收入敏感度, h:利率敏感度）
LM曲线白话版：LM是"货币供给=货币需求"的连线。收入多了→大家需要更多现金→利率必须升高才能让人少持有现金。
什么会让LM右移？央行印更多钱（M↑）→市面上钱变多→利率下降→LM右移。
⚠️ 最容易搞混的地方：
• IS右移 = 财政扩张（政府花钱/减税）→ Y↑, r↑
• LM右移 = 货币扩张（央行印钱）→ Y↑, r↓
记住：财政让利率升，货币让利率降！

IS-LM均衡与政策

均衡点：IS与LM的交点 → 同时决定均衡Y和均衡r。

政策曲线移动Yr
扩张财政IS右移↑↑
扩张货币LM右移↑↓
挤出效应：扩张性财政 → IS右移 → r↑ → 私人投资I↓ → 部分抵消了财政扩张的效果。挤出效应在LM越陡峭时越大。
流动性陷阱：利率极低时，LM接近水平，货币需求对利率无限敏感。此时货币政策几乎无效，但财政政策非常有效。
挤出效应通俗版：政府借钱修路→钱的需求增加→利率上涨→民营企业觉得借钱太贵不投资了→经济没怎么增长。
就像在游泳池里，政府占了一大块位置，私人企业就被挤到角落去了。
🗺️ 这章讲什么？ IS-LM只决定收入Y和利率r，但没考虑物价P。AD-AS模型加入了物价，可以分析：为什么物价会涨？经济为什么会衰退？什么叫"滞胀"？
一句话：AD是"大家想买多少"，AS是"工厂能生产多少"，两者决定产出和价格。

AD曲线（总需求曲线）

从IS-LM推导：价格P↑ → 实际货币供给M/P↓ → LM左移 → r↑ → I↓ → Y↓。所以AD是向右下方倾斜的曲线。

右移因素：扩张性财政政策、扩张性货币政策、消费信心↑、NX↑

AD为什么向下倾斜？物价涨了→手里的钱不值钱了（实际货币少了）→利率涨了→企业不投资了→总收入下降。
生活类比：超市涨价了→你买东西要花更多钱→总消费减少→经济变冷。

SRAS与LRAS

SRAS（短期总供给）：向上倾斜——价格上升短期内刺激企业增加产出。原因：粘性工资、粘性价格、错觉理论。

LRAS（长期总供给）：垂直——长期产出由资本、劳动、技术决定，与价格无关。位置=潜在GDP（自然率产出）。

SRAS为什么向上倾斜？面包涨价了→面包师傅以为赚更多→多生产。但工人的工资是之前谈好的（粘性工资），所以短期成本没变，企业愿意多生产。
LRAS为什么垂直？长期来看，工人会要求涨工资，原料也会涨价。所以涨价只是"水涨船高"，实际产量不变——产出只取决于有多少机器、工人和技术。

冲击分析框架

冲击类型曲线YP
正向需求冲击（AD右移）AD→↑↑
负向供给冲击（SRAS左移）SRAS←↓↑
滞胀：负向供给冲击（如石油涨价）导致SRAS左移 → 产出下降+价格上升 = 停滞+通胀。这无法用单纯的需求管理政策解决，是宏观政策的经典难题。
长期调整：若AD右移使Y > 潜在GDP → 工资上涨 → SRAS左移 → 经济回到潜在GDP但价格更高。扩张政策的长期效果只是推高价格。
⚠️ 考试最爱考的对比：
• 需求冲击（AD移动）：Y和P同方向变——要涨都涨，要跌都跌
• 供给冲击（AS移动）：Y和P反方向变——这是"滞胀"的根源！
记忆："需求拉涨一起涨，供给出问题一升一降"

失业的三种类型

类型定义原因
摩擦性失业找工作的过渡期失业信息不对称、求职匹配需要时间
结构性失业技能与岗位不匹配技术变革、产业转移
周期性失业经济衰退期失业总需求不足
自然失业率 = 摩擦性失业率 + 结构性失业率

通货膨胀的类型

需求拉动型通胀：总需求过度增长 → AD右移 → "过多的货币追逐过少的商品"。

成本推动型通胀：生产成本上升 → SRAS左移 → 价格上升但产出下降。

惯性通胀：预期通胀自我实现——人们根据过去通胀预期未来通胀，工资和价格螺旋上升。

菲利普斯曲线

π = πe − β (u − un)

短期：失业与通胀存在替代关系（SRPC：负斜率）。
长期：失业回到自然率，菲利普斯曲线垂直（LRPC：u = un）。

政策含义：短期内可以通过扩张政策降低失业（代价是通胀上升）。长期看，试图将失业压到自然率以下只会导致通胀不断加速（加速主义假说），失业最终必然回到un。

产出缺口 = 实际GDP − 潜在GDP。正值表示经济过热，负值表示资源闲置。

国际收支与汇率

经常账户：贸易差额（出口−进口）+ 净收入转移。

资本账户：资本流入−流出。

汇率：本币对外币的比价。升值 = 本币更值钱；贬值 = 本币不值钱。

实际汇率：e = E × P/P*（E:名义汇率, P:国内价格, P*:国外价格）。

净出口：NX = X − M = NX₀ − m·Y

蒙代尔-弗莱明模型（核心结论）

假设：小型开放经济 + 完全资本流动（国内利率=世界利率r*）。

汇率制度财政政策货币政策
浮动汇率无效有效
固定汇率有效无效
蒙代尔不可能三角：一国不可能同时实现：①资本自由流动、②固定汇率、③独立货币政策。三者只能选其二。

财政政策工具

政府支出G：直接拉动总需求，乘数效应大。

税收T：减税 → 可支配收入↑ → 消费↑ → Y↑。乘数小于政府支出乘数（因为MPC<1）。

自动稳定器：税收和转移支付自动对冲经济波动（如累进税制、失业保险），无需额外立法。

政策时滞：认识时滞 + 决策时滞 + 执行时滞。财政政策的决策时滞通常更长。

挤出效应复习：财政扩张 → IS右移 → r↑ → I↓，削弱了财政扩张的效果。但在流动性陷阱或固定汇率下，挤出效应可能消失。

货币政策工具

三大传统工具：

  • 公开市场操作：央行买卖国债 → 调节货币供给（最常用、最灵活）
  • 法定准备金率：调整银行必须持有的准备金比例（效果最猛烈，很少用）
  • 再贴现率：央行对商业银行的贷款利率

传导机制：M↑ → r↓ → I↑ → Y↑（利率渠道）

vs 财政政策：货币政策的决策时滞短（央行独立决策），但执行时滞可能长。在流动性陷阱中货币政策失效。固定汇率下货币政策受制于汇率目标。

经济增长的来源

直接来源：资本积累（K↑）、劳动投入（L↑）、技术进步（A↑）。

深层因素：制度（产权保护、法治）、人力资本（教育、健康）、开放度、地理等。

生产函数：Y = A·F(K, L)　或　Y = A·Kα·L1−α

索洛增长模型

核心方程：Δk = s·f(k) − (δ + n + g)k

s·f(k)：人均储蓄/投资。（δ+n+g)k：弥补折旧、人口增长和技术进步的所需投资。

稳态条件：s·f(k*) = (δ + n + g)k*

核心结论：

  • 储蓄率↑ → 稳态k*和y*↑，但长期增长率不变（仍取决于技术进步）
  • 人口增长率↑ → 稳态k*和y*↓（资本被稀释）
  • 技术进步是长期人均产出增长的唯一源泉
应试要点：索洛模型最重要的结论是——储蓄率影响稳态水平但不影响长期增长率。长期增长只能靠技术进步。这区分了"水平效应"和"增长效应"。

内生增长理论

核心思想：技术进步不是"天降"的，而是经济体系内部产生的（研发投入、知识溢出、人力资本积累）。

政策含义：补贴研发、保护知识产权、投资教育→可持续增长，而不只是水平提升。
