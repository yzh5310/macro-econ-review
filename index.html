高频概念速查

GDP 衡量的是一定时期内最终产品和服务的市场价值，但它不反映收入分配、闲暇、环境质量、地下经济和非市场活动，因此不能完全等同于社会福利。
扩张性财政政策增加政府支出，可能推高利率，导致私人投资减少，这种私人投资被政府支出"挤出"的现象称为挤出效应。
当利率已经很低时，人们愿意持有货币而不愿购买债券，货币需求对利率极其敏感，此时增加货币供给可能难以继续降低利率，货币政策效果减弱。
滞胀是经济停滞、高失业与通货膨胀同时存在的状态，通常可由负向供给冲击解释，例如生产成本上升导致 SRAS 左移。
自然失业率包括摩擦性失业和结构性失业，是经济正常运行中仍会存在的失业；周期性失业则与经济衰退和总需求不足有关。

GDP（国内生产总值）

定义：一定时期内一国境内生产的所有最终产品和服务的市场价值总和。

三种核算方法：

  • 支出法：GDP = C（消费）+ I（投资）+ G（政府购买）+ NX（净出口）
  • 收入法：GDP = 工资 + 利息 + 租金 + 利润 + 折旧 + 间接税
  • 生产法：各行业增加值之和
GDP = C + I + G + NX

名义GDP vs 实际GDP

名义GDP：按当年价格计算。
实际GDP：按基年价格计算，剔除价格变动。

GDP平减指数 = (名义GDP / 实际GDP) × 100
应试要点：计算"经济增长率"时用实际GDP，而非名义GDP。平减指数衡量的是全部商品价格变化。

CPI vs GDP平减指数

进口品包含不包含

失业率与奥肯定律

自然失业率：摩擦性失业 + 结构性失业（经济正常运转时的最低失业率）。

周期性失业：经济衰退导致的总需求不足引起的失业。

奥肯定律：实际GDP每低于潜在GDP 2%，失业率上升约1个百分点。即：Δ失业率 ≈ −0.5 × (实际GDP增长率 − 潜在GDP增长率)

消费函数与储蓄函数

凯恩斯消费函数：C = a + b·Yd（a:自主消费, b:边际消费倾向MPC）

MPC：每增加1单位可支配收入，消费增加多少（0 < MPC < 1）

MPS：边际储蓄倾向 = 1 − MPC

C = a + MPC × Yd　　S = −a + (1−MPC) × Yd

均衡收入与乘数

均衡条件：Y = AE = C + I + G + NX（总产出=总需求）

投资乘数：k = 1/(1 − MPC)

政府支出乘数：ΔY/ΔG = 1/(1 − MPC)

税收乘数：ΔY/ΔT = −MPC/(1 − MPC)（注意负号！减税使收入上升）

ΔY = k × ΔI = ΔI / (1 − MPC)
应试要点：MPC越大，乘数越大。MPC=0.8时乘数=5。计算题通常给MPC让你算均衡收入和乘数。注意区分"自主消费a增加"和"MPC变化"的不同效应。

IS曲线（产品市场均衡）

含义：使产品市场均衡（I=S）的所有(Y, r)组合。

斜率：负斜率——利率r上升 → 投资I下降 → 总需求下降 → 均衡收入Y下降。

右移因素：G↑、T↓、自主消费↑、自主投资↑、NX↑

左移因素：G↓、T↑、自主消费↓、自主投资↓、NX↓

LM曲线（货币市场均衡）

含义：使货币市场均衡（L=M/P）的所有(Y, r)组合。

斜率：正斜率——收入Y上升 → 货币需求L↑ → 利率r上升以维持均衡。

右移因素：M↑、P↓（实际货币供给增加）

左移因素：M↓、P↑（实际货币供给减少）

货币需求：L = kY − hr　（k:收入敏感度, h:利率敏感度）

IS-LM均衡与政策

均衡点：IS与LM的交点 → 同时决定均衡Y和均衡r。

政策曲线移动Yr
扩张财政IS右移↑↑
扩张货币LM右移↑↓
挤出效应：扩张性财政 → IS右移 → r↑ → 私人投资I↓ → 部分抵消了财政扩张的效果。挤出效应在LM越陡峭时越大。
流动性陷阱：利率极低时，LM接近水平，货币需求对利率无限敏感。此时货币政策几乎无效，但财政政策非常有效。

AD曲线（总需求曲线）

从IS-LM推导：价格P↑ → 实际货币供给M/P↓ → LM左移 → r↑ → I↓ → Y↓。所以AD是向右下方倾斜的曲线。

右移因素：扩张性财政政策、扩张性货币政策、消费信心↑、NX↑

SRAS与LRAS

SRAS（短期总供给）：向上倾斜——价格上升短期内刺激企业增加产出。原因：粘性工资、粘性价格、错觉理论。

LRAS（长期总供给）：垂直——长期产出由资本、劳动、技术决定，与价格无关。位置=潜在GDP（自然率产出）。

冲击分析框架

冲击类型曲线YP
正向需求冲击（AD右移）AD→↑↑
负向供给冲击（SRAS左移）SRAS←↓↑
滞胀：负向供给冲击（如石油涨价）导致SRAS左移 → 产出下降+价格上升 = 停滞+通胀。这无法用单纯的需求管理政策解决，是宏观政策的经典难题。
长期调整：若AD右移使Y > 潜在GDP → 工资上涨 → SRAS左移 → 经济回到潜在GDP但价格更高。扩张政策的长期效果只是推高价格。

失业的三种类型

类型定义原因
摩擦性失业找工作的过渡期失业信息不对称、求职匹配需要时间
结构性失业技能与岗位不匹配技术变革、产业转移
周期性失业经济衰退期失业总需求不足
自然失业率 = 摩擦性失业率 + 结构性失业率

通货膨胀的类型

需求拉动型通胀：总需求过度增长 → AD右移 → "过多的货币追逐过少的商品"。

成本推动型通胀：生产成本上升 → SRAS左移 → 价格上升但产出下降。

惯性通胀：预期通胀自我实现——人们根据过去通胀预期未来通胀，工资和价格螺旋上升。

菲利普斯曲线

π = πe − β (u − un)

短期：失业与通胀存在替代关系（SRPC：负斜率）。
长期：失业回到自然率，菲利普斯曲线垂直（LRPC：u = un）。

政策含义：短期内可以通过扩张政策降低失业（代价是通胀上升）。长期看，试图将失业压到自然率以下只会导致通胀不断加速（加速主义假说），失业最终必然回到un。

经济周期的四个阶段

繁荣 → 衰退 → 萧条 → 复苏 → （循环往复）。

产出缺口 = 实际GDP − 潜在GDP。正值表示经济过热，负值表示资源闲置。

国际收支与汇率

经常账户：贸易差额（出口−进口）+ 净收入转移。

资本账户：资本流入−流出。

汇率：本币对外币的比价。升值 = 本币更值钱；贬值 = 本币不值钱。

实际汇率：e = E × P/P*（E:名义汇率, P:国内价格, P*:国外价格）。

净出口：NX = X − M = NX₀ − m·Y

蒙代尔-弗莱明模型（核心结论）

假设：小型开放经济 + 完全资本流动（国内利率=世界利率r*）。

汇率制度财政政策货币政策
浮动汇率无效有效
固定汇率有效无效
蒙代尔不可能三角：一国不可能同时实现：①资本自由流动、②固定汇率、③独立货币政策。三者只能选其二。

财政政策工具

政府支出G：直接拉动总需求，乘数效应大。

税收T：减税 → 可支配收入↑ → 消费↑ → Y↑。乘数小于政府支出乘数（因为MPC<1）。

自动稳定器：税收和转移支付自动对冲经济波动（如累进税制、失业保险），无需额外立法。

政策时滞：认识时滞 + 决策时滞 + 执行时滞。财政政策的决策时滞通常更长。

挤出效应复习：财政扩张 → IS右移 → r↑ → I↓，削弱了财政扩张的效果。但在流动性陷阱或固定汇率下，挤出效应可能消失。

货币政策工具

三大传统工具：

  • 公开市场操作：央行买卖国债 → 调节货币供给（最常用、最灵活）
  • 法定准备金率：调整银行必须持有的准备金比例（效果最猛烈，很少用）
  • 再贴现率：央行对商业银行的贷款利率

传导机制：M↑ → r↓ → I↑ → Y↑（利率渠道）

vs 财政政策：货币政策的决策时滞短（央行独立决策），但执行时滞可能长。在流动性陷阱中货币政策失效。固定汇率下货币政策受制于汇率目标。

经济增长的来源

直接来源：资本积累（K↑）、劳动投入（L↑）、技术进步（A↑）。

深层因素：制度（产权保护、法治）、人力资本（教育、健康）、开放度、地理等。

生产函数：Y = A·F(K, L)　或　Y = A·Kα·L1−α

索洛增长模型

核心方程：Δk = s·f(k) − (δ + n + g)k

s·f(k)：人均储蓄/投资。（δ+n+g)k：弥补折旧、人口增长和技术进步的所需投资。

稳态条件：s·f(k*) = (δ + n + g)k*

核心结论：

  • 储蓄率↑ → 稳态k*和y*↑，但长期增长率不变（仍取决于技术进步）
  • 人口增长率↑ → 稳态k*和y*↓（资本被稀释）
  • 技术进步是长期人均产出增长的唯一源泉
应试要点：索洛模型最重要的结论是——储蓄率影响稳态水平但不影响长期增长率。长期增长只能靠技术进步。这区分了"水平效应"和"增长效应"。

内生增长理论

核心思想：技术进步不是"天降"的，而是经济体系内部产生的（研发投入、知识溢出、人力资本积累）。

政策含义：补贴研发、保护知识产权、投资教育→可持续增长，而不只是水平提升。
